Aplicación web hecha con el framework ASP.NET y Visual Studio que tiene como objetivos:

• Diseñar una aplicación a partir del análisis realizado en la asignatura 
    ASM (Análisis de Sistemas Multimedia). El diseño consiste en la 
    especificación de la arquitectura de la aplicación mediante la aplicación 
    de patrones de diseño, la especificación la lógica de negocio mediante 
    un modelo de dominio y la especificación del interfaz de usuario 
    mediante prototipos de interfaz de usuario.

• Programar una aplicación en el entorno .NET. Esto implica seguir el 
    diseño indicado previamente, conocer frameworks para .NET para 
    desarrollar aplicaciones a nivel de lógica de negocio con NHibernate o 
    Entity Framework, y para la interfaz de usuario Web con ASP.MVC o 
    ASP.Net Core 2.
